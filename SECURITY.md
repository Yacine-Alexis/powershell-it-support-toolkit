# Security Policy

## Supported Versions

This repository is a starter toolkit and does not currently publish versioned security support windows.

## Reporting a Vulnerability

If you discover a security issue in one of these scripts:

1. Do not open a public issue with exploit details.
2. Share a private report with the repository owner, including reproduction steps and impact.
3. Allow time for investigation and remediation before public disclosure.

## Scope

Treat stored credentials, exported inventory data, and Active Directory output as sensitive operational data.
